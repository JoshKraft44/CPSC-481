Bus Ticketing Kiosk: Quick Guide

This kiosk helps you easily purchase bus tickets, reload cards, or check schedules.

  1. Select a pass type — Day, Monthly, or Yearly.
  2. Select user category — Adult, Senior, Student, Minor.
  3. Choose quantity and proceed to payment.
  4. Collect pass and receipt. The kiosk resets after a short time.
